# amily

A personal app for keeping up with the people who matter. Log hangouts, track memories, split costs, and use AI to stay thoughtful about your friendships.

**[amily-ten.vercel.app](https://amily-ten.vercel.app/)**

## Stack

- React + TypeScript + Vite
- Supabase — database, auth, edge functions
- Google Gemini — AI features
- Stripe — payments
- Cloudinary — photo storage
- Vercel — hosting
